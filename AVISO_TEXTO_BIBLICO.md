# Aviso sobre el texto bíblico

Biblia Vida incluye la traducción **Reina-Valera 1909**.

La Reina-Valera 1909 está identificada por eBible.org como **Public Domain / Dominio público**. El texto se integra localmente para permitir lectura y búsqueda sin conexión.

Datos verificados en esta compilación:

- 66 libros.
- 1,189 capítulos.
- 31,102 versículos.

Fuentes de referencia:

- https://ebible.org/spaRV1909/
- https://github.com/MaatheusGois/bible/tree/main/versions/es

La interfaz, identidad visual, iconos propios y código específico de Biblia Vida pertenecen al proyecto de Proca Corporación, salvo las bibliotecas de terceros sujetas a sus licencias correspondientes.
