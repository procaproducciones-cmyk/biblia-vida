BIBLIA VIDA v1.2.4
==================

Estado: proyecto corregido y listo para compilar mediante GitHub Actions.

CAUSA ENCONTRADA
----------------
1. El proyecto fuente anterior no contenía www/data/biblia-oso1569.json.
2. La versión 1.2.3 podía guardar en caché la respuesta HTTP 404 de ese archivo.
3. Al actualizar, el service worker podía seguir entregando ese error antiguo.

CORRECCIONES INTEGRADAS
-----------------------
- Descarga automática de la fuente completa de la Biblia del Oso durante la compilación.
- Descarga alternativa de los 66 libros por separado si falla el archivo completo.
- Conversión y validación obligatoria: 66 libros, 1,189 capítulos y más de 30,000 versículos.
- Copia física del archivo final dentro de los recursos Android y dentro del APK.
- Verificación del contenido del APK antes de publicar el artefacto.
- Caché actualizado a biblia-vida-v1.2.4.
- Las respuestas HTTP fallidas ya no se guardan en caché.
- El service worker se desactiva en la aplicación Android para evitar cachés antiguos.
- versionCode 7 / versionName 1.2.4.

PASO IMPORTANTE PARA LA PRUEBA
-----------------------------
Desinstala la versión 1.2.3 del teléfono antes de instalar la 1.2.4. La desinstalación elimina el caché 404 creado por la versión anterior.

PARA GENERAR LA APK
-------------------
1. Sube todo el contenido del proyecto a GitHub, incluida la carpeta .github.
2. Abre Actions > Generar APK Biblia Vida.
3. Ejecuta el flujo y espera la marca verde.
4. Descarga el artefacto Biblia-Vida-APK.
5. Instala Biblia-Vida-v1.2.4.apk.
