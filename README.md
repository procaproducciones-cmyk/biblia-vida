# 📖 Biblia Vida v1.0.0

Aplicación Android de la **Biblia Reina-Valera 1909**, diseñada para funcionar completamente **sin conexión**, **sin publicidad**, **sin cuentas** y **sin recopilación de datos personales**.

Proyecto desarrollado para **Proca Corporación**.

## Funciones incluidas

- Biblia completa: 66 libros, 1,189 capítulos y 31,102 versículos.
- Lectura por libro y capítulo.
- Navegación al capítulo anterior y siguiente.
- Buscador bíblico local por palabras o frases.
- Versículo del día.
- Favoritos.
- Notas personales por versículo.
- Copiar y compartir versículos.
- Modo claro, oscuro o igual al sistema.
- Tamaño de letra y espaciado configurables.
- Guarda automáticamente la última lectura.
- Opción para mantener la pantalla encendida durante la lectura.
- Diseño adaptable para teléfonos Android.
- Sin permiso de Internet en Android.

## Tecnología

- HTML, CSS y JavaScript.
- Capacitor 8.
- Proyecto Android nativo incluido.
- Datos bíblicos almacenados localmente en JSON.
- Preferencias, favoritos y notas almacenados localmente en el dispositivo.

## Abrir el proyecto

Consulta **GUIA_GENERAR_APK.md**. El proyecto ya incluye la plataforma Android, el identificador `com.proca.bibliavida`, iconos, splash y la versión 1.0.0.

## Texto bíblico

La aplicación usa la **Reina-Valera 1909**, indicada como texto de dominio público. Consulta **AVISO_TEXTO_BIBLICO.md**.

## Privacidad

Consulta **PRIVACIDAD.md**. La versión 1.0.0 no incluye publicidad, analítica, cuentas, servidores propios ni permiso Android de Internet.
